Getting started

Create an account

To get started in nuntium you first need to create an account which will manage the organizations, publishers, bundles and everything else.

Create an organization

An organization acts as a container for the publishers belonging to it.

You can create a new one in Settings / Organizations.

Create a publisher

One organization can have multiple publishers.

To create a new publisher go to Settings / Organizations / organization / Publishers.

Verify the publisher's domain

Once a new publisher is created the first thing you should do is verifying you own the domain you specified when creating it.

This step is required in order to create articles.

Add an author

Create an invite

To add authors to a publisher you must first invite them.

To invite a user to join as an author go to Settings / Organizations / organization / Publishers / publisher / Authors / Invite.

Wait for the user to accept the invite

Once invited a user will receive an email asking him to accept the invite. He will become an author only if and when he accepts it.

Note that an invite expires after 7 days.

Publish an article

Create the first draft

To create an article an author must first create the first draft.

A draft can be edited as many times as needed before submitting it for verification.
